# KNOW/Padrinho/raw-data/ — Material RAW do Padrinho

Originais crus que alimentam o conhecimento do Padrinho (ex.: transcrições de founders, pesquisa, documentos enviados). **Vazio por ora.**

## Convenção (mesma do CTBM)
- Aqui ficam **apenas os originais** (PDFs, `.txt`, etc.) — não editar.
- A **transcrição verbatim** de cada original vira um canônico `../KNOW_<Tema>.md` (sem resumo, sem interpretação).
- Dali o conteúdo é derivado nos formatos. Ver `POSTS/Padrinho/STRATEGY_Padrinho.md` (§ atomização).
