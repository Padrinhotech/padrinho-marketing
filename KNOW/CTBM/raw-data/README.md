# KNOW/CTBM/raw-data/ — Material RAW enviado pela CTBM

Esta pasta guarda **apenas o material bruto original** que a CTBM nos envia: os **PDFs** (e a extração `.txt` de máquina). É a fonte de verdade factual.

> ⚠️ **O conteúdo transcrito NÃO fica aqui.** As transcrições verbatim são **canônicas** e vivem em `KNOW/CTBM/` (`KNOW_CTBM_<Tema>.md`). Aqui ficam só os arquivos crus.

## O que vive aqui

| Arquivo | O que é |
|---|---|
| `CTBM_RAW_*.pdf` | **Original** enviado pela CTBM (não editar) |
| `CTBM_RAW_*.txt` | Texto extraído do PDF (artefato de máquina) |

## Regras

1. **Não editar os originais.** O PDF é a fonte de verdade factual (datas, nomes, horários, números — ex.: *Pet Terapia: sextas-feiras às 14h* · *Programa de Família: terças e quintas*).
2. A **transcrição verbatim** de cada PDF é feita em `../KNOW_CTBM_<Tema>.md` — **sem resumo, sem interpretação** (só normalização de artefatos de extração: ligaduras `ﬁ`/`ﬂ`, espaços quebrados, hifenização partida).
3. Dali, o Padrinho deriva os formatos (Blog âncora → Reels · Carrossel(+FB) · Canais Internos · LinkedIn; Newsletter mensal). Ver `POSTS/CTBM/STRATEGY_CTBM.md` §5.1.
