# Marketing — Estratégia e Criação de Conteúdo

Esta pasta contém toda a lógica de marketing do Padrinho.app:
estratégia de conteúdo, personas, pilares editoriais, brand guidelines,
referências visuais e posts gerados.

## Estrutura

```
marketing/
├── KNOW/              ← Knowledge: brand positioning, market context, personas, editorial pillars
│   ├── KNOW_BrandPositioning.md
│   ├── KNOW_MarketContext.md       ← Updated by Market Agent
│   ├── KNOW_EditorialPillars.md
│   ├── KNOW_TrendRadar.md
│   ├── KNOW_ICP.md
│   ├── KNOW_UserInsights.md        ← Updated by User Insights Agent
│   └── KNOW_[PersonaName].md       ← 4 personas
│
├── SKILL/             ← Skills: copy rules, component system, photo guidelines, etc
│   ├── SKILL_Documentation.md      ← Master documentation standard
│   ├── SKILL_CopyRules.md
│   ├── SKILL_ComponentSystem.md
│   ├── SKILL_ComponentIndex.md
│   ├── SKILL_FigmaDelivery.md
│   ├── SKILL_PhotoGuidelines.md
│   ├── SKILL_VisualAgent.md
│   ├── SKILL_AudiencePipeline.md
│   ├── SKILL_CampaignSetup.md
│   └── inject-photos.js / .node.js
│
├── AUT/               ← Automations: all 7 agent instructions
│   ├── AUT_InsightsAgent.md
│   ├── AUT_MarketAgent.md
│   ├── AUT_UserInsightsAgent.md
│   ├── AUT_StrategyAgent.md
│   ├── AUT_TacticAgent.md
│   ├── AUT_OperationalAgent.md
│   └── AUT_FigmaAgent.md
│
├── brand/             ← Brand assets: design tokens, product spec
├── insights/          ← Daily insights output
└── posts/             ← Generated social content (Figma Agent output)
```
