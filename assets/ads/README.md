# Ad Creatives — Padrinho App

Public folder for ad image assets. Files here are referenced by raw.githubusercontent.com URLs in Meta ad creative uploads.

## Usage
Raw URL pattern:
```
https://raw.githubusercontent.com/Padrinhotech/padrinho-marketing/main/assets/ads/<filename>.png
```

## Export Instructions (from Figma)
1. Open https://www.figma.com/design/sBItPeNLyvT5EMyKLqQbRv
2. Select each node above
3. Export as PNG @ 2x
4. Name files as `ad-01.png` through `ad-06.png`
5. Commit to this folder
6. Claude will auto-upload to Meta using raw.githubusercontent.com URLs

## Meta Image Requirements
- Format: PNG or JPG
- Recommended: 1080x1080 (feed) or 1080x1920 (stories)
- Max size: 30MB
- Text overlay: <20% of image area
