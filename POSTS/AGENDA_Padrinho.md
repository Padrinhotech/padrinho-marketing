# AGENDA Padrinho — Temas H2 2026

Calendário editorial centralizado. **Gerador de verdade única** para planejamento estratégico.

Estrutura: data | tema | pilar editorial | persona primária | canais | status

---

## Metodologia

- **Semana:** Cada linha = 1 semana (seg-dom) com **4 posts** (Sun/Mon/Wed/Fri)
- **Tema:** Tema único para a semana; **4 ângulos diferentes** para cada post
- **Pilar:** Reconhecimento | Desmascaramento | Acolhimento | Empoderamento | Reingresso
- **Persona:** Rosa (Equilibrista) | Pedro (Autônomo Solitário) — foco principal; 2 posts Rosa + 2 posts Pedro por semana
- **Canais:** Blog | IG Carousel | IG Reshare | WhatsApp | LinkedIn (x4 por semana)
- **Status:** Planejado | Em Tática | Em Operação | Publicado | Analisado
- **Estrutura de Pastas:**
  ```
  POSTS/WEEK01_DDMMYY_TemaSemana/
    01_SUNDAY_AnglePerson/
      POST_Overview.md
      blog-post.md
      instagram-carousel.md (or Figma link)
      instagram-reshare.md
      whatsapp-text.md
      linkedin-copy.md
    02_MONDAY_AnglePerson/
    03_WEDNESDAY_AnglePerson/
    04_FRIDAY_AnglePerson/
  ```

## Como Usar Esta Agenda

### Para AGENT_Strategy
1. Leia a linha da semana
2. Crie `POSTS/DDMMYYYY_Tema/POST_Overview.md`
3. Inclua: pergunta central, pilar, persona, refs de KNOW_

### Para AGENT_Tactic
1. Leia POST_Overview
2. Crie 6 arquivos de TEXTO (Blog, News, IG Captions, Podcast, WhatsApp, LinkedIn)
3. Cada um otimizado para o canal

### Para AGENT_Operational
1. Valida os 6 textos
2. Commit para GitHub
3. Sinaliza para Figma

### Para AGENT_Figma
1. Lê POST_Overview + instagram-captions.md
2. Cria imagens (Camada 2)
3. Exporta para assets/images/
