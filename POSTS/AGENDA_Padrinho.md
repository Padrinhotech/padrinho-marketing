## Como Usar Esta Agenda

### Para AGENT_Strategy
1. Leia a linha da semana
2. Crie `POSTS/DDMMYYYY_Tema/POST_Overview.md`
3. Inclua: pergunta central, pilar, persona, refs de KNOW_

### Para AGENT_Tactic
1. Leia POST_Overview
2. Crie 6 arquivos de TEXTO (Blog, News, IG Captions, Podcast, WhatsApp, LinkedIn)
3. Cada um otimizado para o canal

### Para AGENT_Operational
1. Valida os 6 textos
2. Commit para GitHub
3. Sinaliza para Figma

### Para AGENT_Figma
1. Lê POST_Overview + instagram-captions.md
2. Cria imagens (Camada 2)
3. Exporta para assets/images/
